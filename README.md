# CreatorConnect

A web platform that enables brands to discover, evaluate, and collaborate directly with influencers — no middlemen, no guesswork.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js *(in progress)*
- **Database:** MySQL
- **Version Control:** Git & GitHub

## Features (Phase 1 — Complete)

- Landing page with How It Works section
- User signup with role selection (Brand / Influencer)
- User login with form validation
- Influencer profile form (niche, metrics, social handles, collaboration preferences)
- Brand profile form (industry, campaign preferences, budget)
- Influencer search page with live filtering by niche, followers, engagement rate, and budget

## Planned Features (Phase 2+)

- REST API with Node.js and Express
- MySQL database integration
- JWT-based authentication
- Role-based dashboards
- Collaboration request system
- Messaging and notifications
